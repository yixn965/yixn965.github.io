# Jing Gao Personal Homepage

This folder contains a static GitHub Pages homepage for `yixn965.github.io`.

## Publish

1. Create a GitHub repository named `yixn965.github.io`.
2. Copy these files into that repository.
3. Push to the `main` branch.
4. In GitHub, open `Settings -> Pages`, select `Deploy from a branch`, then choose `main` and `/root`.

The page will be available at:

```text
https://yixn965.github.io/
```
